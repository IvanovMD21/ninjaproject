# ninjaproject
Игра была написана с помощью ИИ Deepseek.
Изначально была проработана идея и план действий, а затем уже с помощь ИИ постепенно выполне-
ние пунктов плана.
Много приходилось искать ошибок и исправлять самому, т.к. Deepseek периодически не справлялся.

Выполнение всего проекта заняло 1.5 недели, доработки были не каждый день, если считать непос-
редственно полезное время работы, то на проект ушло +-30 часов.

Использовались сгенерированые персонажи, объекты. В ручную проработаны спрайты персонажей и 
некоторых объектов для анимации движения и прыжков с помощью Adobe Photoshop.
Подключен стилизованый шрифт.

Не знаю точно по срокам сдачи задания, но планов и желания доработки еще много, например:
1. добавить восточную музыку
2. анимацию смерти врагов
3. анимацию получения доп. вооружения
и др.
